# FilmTrade

Cinema Intelligence Platform

A premium web platform that allows investors to discover film opportunities,
producers to manage projects, and administrators to monitor platform activity.

## Features

• Investor Dashboard
• Producer Dashboard
• Admin Dashboard
• Discover Movies
• Authentication
• Watchlist
• Portfolio
• Analytics

## Tech Stack

Next.js
TypeScript
Tailwind CSS
React
Local Storage
Prisma (planned)
